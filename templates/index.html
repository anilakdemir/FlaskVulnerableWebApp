Zayıf Web Uygulaması


Zayıf Web Uygulaması Nedir?

Zayıf Web Uygulaması kasıtlı olarak bırakılmış güvenlik zafiyetleri içeren bir Python/SQLite web uygulamasıdır. Kasıtlı olarak bırakılmış güvenlik zafiyetlerini içeren zayıf web uygulamaları, uygulamalı siber güvenlik eğitimlerinde önemli bir rol oynamaktadır. Öğrenciler yasal olmayan bir şekilde gerçek sistemlere saldırmadan bu tip uygulamalar üzerinde öğrendiklerini tecrübe ederek kendilerini geliştirebilmektedirler.

Bu Projenin Amacı

Farklı programlama dillerinde (PHP, ASP, C#.NET, Node.js, PERL, Java, Ruby on Rails, Python, C++, ColdFusion vb.) yazılmış zayıf web uygulamaları bulunmaktadır. Fakat, Türkçe yazılmış ve iyi bir şekilde dökümantasyonu yapılmış web uygulamaları yok denecek kadar azdır.
