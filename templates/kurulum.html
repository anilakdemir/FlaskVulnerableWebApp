KURULUM


Kullanılan Teknolojiler/Uygulamalar

  • Python 2.7
  • SQLite
  • virtualenv
  • Flask Micro-Framework
  • Jinja2 Templating Engine

Kurulum Adımları

  • Uygulama Windows, GNU/Linux ve MacOS işletim sistemlerinde çalışabilir. Aşağıda anlatılan kurulum MacOS içindir. Diğer işletim sistemlerinde farklılık gösterebilir.
  • Python dili ile geliştirme yapabilmek için aşağıdaki linkten Python 2.7 indirip kulumunu yapınız.
  • Python 2.7 indirmek için tıklayınız.
  • Kurulumu tamamladıktan sonra terminali açınız.
  • Terminale 'python' yazarak kurulumun başarılı olduğunu kontrol ediniz.


  • Yukarıdaki gibi bir görüntüyle karşılaşıyorsanız kurulum başarılı demektir. 'exit()' komutu ile çıkış yapabilirsiniz.
  • Terminale 'pip install virtualenv' yazarak virtualenv kurulumunu gerçekleştiriniz.
  • Kurulum gerçekleştikten sonra Terminalden proje dosyasının yoluna geliniz. ( Terminalde 'cd' komutunu kullanabilirsiniz.)
  • Ortamı aktifleştirmek için terminale dosyanın içerisindeyken 'source env/bin/activate' komutunu giriniz.
  • Aşağıdaki gibi bir görüntüyle karşılaşıyorsanız ortamı aktif ettiniz demektir. Satıra '(env)' ifadesi eklenir.


  • Virtualenv başarıyla aktif edildikten sonra, terminale 'pip install flask' komutunu girerek Flask için gerekli modüllerin kurulmasını sağlayabilirsiniz. Jinja2 ve SQLite3 Flask ile birlikte kurulu olarak gelecektir.
  • Son olarak projeyi çalıştırmak için terminale dosya dizininde iken 'python app.py' yazmanız yeterli olacaktır.
  • Projeye 'http://localhost:8080/' adresinden ulaşabilirsiniz.
  • Uyarı: Projenin çalışması için terminali kapatmamalısınız.

Veritabanında Gerekli Tabloların Oluşturulması

  • Uygulamada veritabanı olarak SQLite kullanılmıştır.
  • Uygulama dizini içerisinde örnek veritabanı bulunmaktadır. Silip yeniden oluşturmak istenildiği takdirde aşağıdaki adımlar takip edilebilir.
  • Uygulama çalıştığında uygulamanın bulunduğu dizinde veritabanı 'database.db' ismiyle otomatik olarak oluşmaktadır.
  • Uygulama kullanılmadan önce oluşan veritabanında gerekli tablolar oluşturulmalıdır. Aksi halde 'SQL Enjeksiyon' ve 'XSS' sayfalarında hata ile karşılaşılacaktır.
  • Tabloları oluşturma işlemi için 3. parti uygulama kullanımı tavsiye edilmektedir. (DB Browser for SQLite)
  • Uygulamanın arayüzü aşağıdaki gibidir.


  • Arayüzden 'Open Database' seçeneğini seçerek uygulama dizinindeki 'database.db' veritabanı açılır.


  • Uygulamanın 'File' sekmesinden önce 'Import' seçeneği, ardından 'Database from SQL File...' seçeneği seçilir.


  • Açılan pencereden uygulama dizini içerisindeki 'veritabani.sql' dosyası seçilir ve gelen seçime 'Hayır' yanıtı verilir.


  • Son olarak yapılan işlemlerin veritabanın kayıt edilmesi için arayüzden 'Write Changes' butonuna tıklanır.
  • Uygulama için gerekli olan tabloların başarıyla oluşturulduğundan emin olunduktan sonra uygulama kullanılmaya başlanabilir.
